<h2 id="started">🚀 Iniciando</h2>

<h3>Pré-requisitos</h3>

- [NodeJS](https://github.com/)

<h3>Clonando</h3>

Como clonar o projeto:

```bash
git clone https://github.com/ivanfrancajunior/fast-pizza.git
```

<h3>Iniciando</h3>

Como iniciar o projeto:

```shell
cd fast-pizza
npm install
npm run dev
```
